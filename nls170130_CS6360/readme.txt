							Database project : Library Management System
---------------------------------------------------------------------------------------------------------------------------------------
1. This project works with a GUI interface.

2. Front-end(GUI) is HTML and CSS. Server scripting is done in PHP and Back-end database used is MySQL.

3. Local Server Used is XAMPP Server which has both PHP and MySQL built into it.

4. Download the XAMPP Server from following link:
https://www.apachefriends.org/download.html

5. The minimum system requirements are:
	Operating Systems: Windows/MAC
	Apache: 2.4.33 
	MySQL: 5.6.34
	PHP: 7.2.5
	
6. Install XAMPP Server using xampp.exe

7. In preferences click on Ports tab and set Apache port to 80 and MySQL port to 3306.

8. Default Document root is C:\XAMPP\htdocs.

9. Click on OK and then Click on Start Servers. Both Apache Server and MySQL Server will get started.

10. Copy all the code files into the path C:\MAMP\htdocs.

11. Used Server name : localhost, Username : root, Password : Amplifier!500, Database : books

12. Open http://localhost/phpMyAdmin/? Click on Database tab->Import Button.

13. Browse for the library.sql file provided in the zip folder and import the database;

14. All the tables will be copied to the books database.

15. Typing localhost in the browser opens the homepage of the Library Management System from where we can navigate between the pages.
